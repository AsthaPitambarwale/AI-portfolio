1️⃣ Explain the end-to-end request flow in your project.

Answer:

User enters a question in React frontend.

Frontend sends POST request to /chat endpoint.

3️⃣ How does FastAPI dependency injection (Depends) work?

Answer:

FastAPI uses Depends() to inject dependencies like database sessions.

Example:

def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()

FastAPI automatically creates and closes DB sessions per request.

5️⃣ Difference between 401, 403, 404, 422?

Answer:

401 → Unauthorized (Missing/Invalid authentication)

403 → Forbidden (Authenticated but no permission)

404 → Resource not found

422 → Validation error (Wrong JSON format)

You faced 401 due to missing Authorization header.

Answer:

It creates database tables based on SQLAlchemy models if they don’t exist.

🔟 What is SQLAlchemy ORM?

Answer:

ORM = Object Relational Mapping.

It lets you interact with database using Python objects instead of raw SQL.

Example:

db.query(Resume).all()

Instead of:

SELECT * FROM resume;

🔥 1️⃣ Explain Your Project
❓ Q1: Explain your AI Portfolio project.

Answer:

I built an AI-powered portfolio assistant using React and FastAPI.
Instead of a static portfolio website, users can ask questions about my skills, experience, and projects in natural language.
The frontend is built with React, the backend uses FastAPI, and it integrates with OpenRouter’s LLM API to generate intelligent responses based only on my resume data.
The system is deployed on Render and uses environment variables for secure API key management.

🔥 2️⃣ Architecture Question
❓ Q2: Explain the complete architecture.

Answer:

The architecture follows a client-server model.

User types a question in React frontend.

React sends a POST request to the FastAPI /chat endpoint.

FastAPI processes the message and calls the AI service layer.

AI service builds a structured prompt using resume data.

Backend sends request to OpenRouter API.

OpenRouter returns the AI-generated response.

Backend sends the response back to frontend.

React displays the AI reply in chat UI.

🔥 5️⃣ What is 401 Error?
❓ Q5: You faced a 401 error. What is it and how did you fix it?

Answer:

A 401 error means Unauthorized.
In my case, OpenRouter was rejecting requests because the Authorization header was either missing or incorrect.

I fixed it by:

Ensuring OPENROUTER_API_KEY was properly set in Render environment variables.

Sending the correct header format:

headers = {
  "Authorization": f"Bearer {OPENROUTER_API_KEY}",
  "Content-Type": "application/json"
}

After verifying via /test-key, it worked correctly.

🔥 6️⃣ What is 422 Error?
❓ Q6: Why did you get 422 error?

Answer:

422 means Unprocessable Entity.
It happened because the JSON body sent from Swagger was invalid.

Instead of:

{ "what is skills?" }

It should be:

{ "message": "What are your skills?" }
